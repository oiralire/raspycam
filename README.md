# raspycam
1) **Purpose**: User-friendly Python wrapper for controlling photo still, timelapse and video capture using a Raspberry Pi and an Arducam module.
2) **Aspiration**: Fully hackable DIY compact camera hardware design with support for the full range of officially supported sensor boards including modular lenses

## Outline
- User-friendly Python wrapper for controlling photo still, timelapse and video capture.
- Web app for live preview and triggering scripts
- Client-side capture in the app
- **Future**: Fully hackable DIY compact camera hardware design with support for the full range of officially supported sensor boards including modular lenses

### Phasing
